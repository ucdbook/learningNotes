### Webpack

webpack 是一个现代 JavaScript 应用程序的静态模块打包器\(module bundler\)

**它的功能：**

1. 通过loader、plugins等配置进行模块代码打包

2. 构建一个模块依赖关系图

3. 通过webpack-dev-server启动本地服务

**四个核心概念：**

一、入口\(entry\):

webpack 应该使用哪个模块，来作为构建其内部依赖图的开始。进入入口起点后，webpack 会找出有哪些模块和库是入口起点（直接和间接）依赖的。

二、输出\(output\)



1. loader
2. 插件\(plugins\)
